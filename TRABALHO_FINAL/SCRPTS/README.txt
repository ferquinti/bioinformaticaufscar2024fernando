Scripts: 
    criar_tabela_fernando.py:
        Contem 4 funções:
            Para criar uma tabela vazia, preencher ela com dados aleatórios, zerar as células e exportar em tsv.
    rarefação_fernando.py:
        Uma função para rarefar todas as colunas da tabela (input).
    tss_fernando.py:
             Contém a função para normalizar a tabela, em que o input é a própria tabela gerada no arquivo main.py
    shannon_fernando.py:
        Contém duas funções, uma para calcular o shannon por coluna e a outra com um "apply" para aplicar em todas as colunas da tabela.
    curva_coletor_fernando.py:
        Contém três funções, sendo uma para amostrar os reads, a outra para suavizar a curva e a última para plotar as curvas.
    main.py:
        utiliza todas as funções criadas para geração de dados e analises
